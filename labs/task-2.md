# Задание: Вычислитель суммы аргументов

## Описание
- Создайте в корне проекта (src) пакет ru.hse.java2026
- Создайте класс `SumCalculator`, который запускается из командной строки, обрабатывает все переданные аргументы как строки с числами (возможно, разделёнными пробельными символами), суммирует все целые числа из них и выводит итоговую сумму в консоль.

## Примеры использования
```
java SumCalculator 5 10 15
Вывод: 30

java SumCalculator 7 -4 2
Вывод: 5

java SumCalculator "8 12 -5"
Вывод: 15

java SumCalculator "20 30" " -10"
Вывод: 40

java SumCalculator "  "
Вывод: 0
```

## Требования к обработке
- Аргументы могут содержать цифры, знаки `+` и `-`
- В аргументах возможны любые пробельные символы (пробелы, табы и др.)
- Все числа из всех аргументов нужно найти и просуммировать
- Для хранения чисел и промежуточных итогов используйте `int`
- Отладочные сообщения выводите через `System.err`
