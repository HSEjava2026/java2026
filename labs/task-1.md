# Лабораторная работа: Работа с пакетами Java и сборкой проекта

## Уровень 1: Базовая структура пакета

**Задание:**
1. Создайте в корне проекта пакет `ru.hse.lab1`
2. В пакете создайте класс `Hello.java` с программой Hello World
3. Скомпилируйте и запустите программу

**Ожидаемый результат:**
```
Hello, World!
```

**Структура проекта:**
```
 ru/
  └── hse/
      └── lab1/
          └── Hello.java
```

***

## Уровень 2: Работа с подпакетами и классами

**Задание:**
1. В пакете `ru.hse.lab1` создайте подпакет `second`
2. В пакете `second` создайте класс `Person` с полями `name` и `age`
3. Создайте класс `Main` в пакете `ru.hse.lab1`
4. В `Main` создайте объект `Person` и выведите его информацию

**Требования:**
- Корректные `package` и `import` во всех файлах
- Конструктор для `Person`

**Ожидаемый результат:**
```
Имя: Иван
Возраст: 20
```

**Структура проекта:**
```
ru/
  └── hse/
      └── lab1/
          ├── Main.java
          └── second/
              └── Person.java
```

***

## Уровень 3: Генерация Javadoc

**Задание:**
1. Добавьте Javadoc-комментарии ко всем классам:
    - `@author` — ваша ФИО
    - `@version` — 1.0
    - `@since` — 2026
2. Сгенерируйте документацию в папку `docs`

**Требования:**
```
docs/
├── index.html
├── ru/
│   └── hse/
│       └── lab1/
│           ├── Main.html
│           └── second/
│               └── Person.html
└── ...
```

***

## Уровень 4: Настройка выходной директории

**Задание:**
1. Настройте сборку так, чтобы скомпилированные `.class` файлы попадали в папку `out`
2. Запустите программу из папки `out`

**Ожидаемый результат структуры:**
```

├── out/           # скомпилированные классы
│   └── ru/
│       └── hse/
│           └── lab1/
│               ├── Main.class
│               └── second/
│                   └── Person.class
├── docs/          # javadoc
└── ...
```



***


## Критерии приемки

| Уровень | ✓ Критерий |
|---------|------------|
| **1** | ✅ Пакет `ru.hse.lab1`, `Hello.java`, компиляция/запуск |
| **2** | ✅ Подпакет `second`, `Person`, `Main`, корректные импорты |
| **3** | ✅ Javadoc во всех файлах, документация в `docs/` |
| **4** | ✅ `.class` файлы в `out/`, запуск из `out/` |
